# assignment-2
Interactive portfolio web app with dynamic content, data handling, and AI-assisted features.

## Features


- Personalized greeting (time of day + stored username)
- Section navigation (About / Projects / Contact) with smooth transitions
- Project list with live search, filter by tag, sort by title/date
- Collapsible project details
- Theme (light/dark) saved to `localStorage`
- Fetch example (quotes API) with loading & error states + retry
- Form with validation and success animation
- Accessible ARIA attributes for interactive elements
